# aprog

Допоміжна бібліотека для створення Properties.

### Version

v1.1.3

### Installation

Aprog requires PHP >= 8.0.

```shell
composer require uaproger/aprog
```

### Basic Usage

```shell
php artisan make:property <name>
```
